commutePollute setup instructions

  1. goto finalyear.loosleyweb.co.uk
  2. register for a new account
  3. once the new account is setup and you are logged in goto the sensor page of the website or click here
  4. click on add new sensor
  5. connect to the hotspot generated by the node that is called commutePollute and has the password commutePollute
  6. navigate to the page when prompted or click here, this should then bring you to the configuration page
  7. On the configuration page there should be 2 boxes, one called EUI and the other called appKey.
    • Copy the contents of both into the corresponding boxes on the device configuration page
  8. enter your wifi details and add a password then reset the node
